# RegionsCatalog

## Задание:

Создать веб сервис (без пользовательского веб интерфейса) используя ASP.NET реализующее справочник регионов (свойства: идентификатор, наименование, сокращённое наименование), предоставляющее REST-API на чтение и изменение справочника, справочник должен храниться в БД.
